# Frontend Audit Report: Osamu Dashboard

This audit identifies hardcoded elements, non-functional placeholders ("text-only" features), and user experience gaps within the current frontend implementation.

## 1. Dashboard Overview ([page.tsx](file:///c:/blackcurrent-project/app/%28dashboard%29/page.tsx))

### Hardcoded Data
- **User Role**: Defaults to "Academic Lead" (line 23).
- **Institution**: Defaults to "Global Academy" if not found in auth (line 25).
- **Footer**: The copyright year is dynamic, but the attribution is static.

### Broken/Incomplete UX
- **Mobile Navigation**: The sidebar uses `hidden md:flex`, making it completely invisible on mobile devices. **CRITICAL**: There is currently no "Hamburger" menu button to open the sidebar on small screens.

---

## 2. Dashboard Sub-Pages

### Home View ([Home.tsx](file:///c:/blackcurrent-project/components/dashboard/pages/Home.tsx))
- **Hardcoded Stats**: "Active Viva: 04" and "Graded: 128" are static placeholders.
- **Hardcoded Timeline**: The calendar view shows fixed dates ("Nov 18 — Nov 25, 2023").
- **Daily Pulse**: All events (e.g., "Senior Thesis Review") are hardcoded examples.
- **Text-Only Features**:
    - **Open Full Scheduler**: Styled button with no link/action.
    - **Timeline Navigation**: Chevron buttons are visual only.

### Assignments Hub ([Assignments.tsx](file:///c:/blackcurrent-project/components/dashboard/pages/Assignments.tsx))
- **Hardcoded Files**: "Reference_Doc.pdf" (2.4 MB) is a static UI element.
- **Hardcoded History**: All student records in the grading table (Student A-D) are test data.
- **Text-Only Features**:
    - **Upload Area**: "Browse Documents" button has no file picker trigger.
    - **Grading Constraints**: The textarea for instructions is not hooked up to any "Save" API.
    - **Search Bar**: The "Search students..." input has no filtering logic.
    - **Batch Actions**: The "Download" and "Archive" buttons in the table are non-functional.
    - **Pagination**: "Page 1 of 1" and the accompanying arrows are disabled.

### Viva-Voce Hub ([VivaVoce.tsx](file:///c:/blackcurrent-project/components/dashboard/pages/VivaVoce.tsx))
- **Hardcoded AI Content**: Questions generated by the "Smart Assistant" are hardcoded strings.
- **Hardcoded Media**: The audio player shows a fixed timestamp ("08:42 / 24:00") and a static 40% progress bar.
- **Hardcoded Metrics**: "Technical Depth", "Reflective Log", and "Articulation" scores are fixed.
- **Text-Only Features**:
    - **Initialize Session**: The primary action button is non-functional.
    - **FAB**: The "Start New Viva" floating button has no click handler.
    - **Playback**: Play/Pause icons in the history section are purely decorative.

---

## 3. Global Navigation ([Shared.tsx](file:///c:/blackcurrent-project/components/dashboard/Shared.tsx))
- **Logo Text**: "Academic Excellence" is hardcoded.
- **TopBar Search**: "Search resources..." input is non-functional.
- **Notifications**: The bell icon and profile icon next to it are static.

---

## 4. Authentication Surface

### Entry Pages ([login](file:///c:/blackcurrent-project/app/login/page.tsx) & [signup](file:///c:/blackcurrent-project/app/signup/page.tsx))
- **External Images**: The login background uses a hardcoded Google Photos URL.
- **Text-Only Features**:
    - **Forgot Credentials**: Non-functional link.
    - **Social Auth**: "Google Workspace" and "SSO" buttons are styled placeholders.
    - **Legal Links**: "Terms" and "Privacy Policy" links point to `#`.
    - **Persistence**: "Remember access" checkbox is not connected to a persistent cookies/storage logic.
